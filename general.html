General design rules

The favicon used for this styleguide is found: here
For customer and external webpages use car-favicon found: here
For admin and internal webpages use pen-favicon found: here
A link or reference should if possible be available and visible for favicons or pictures used.

The following general design topics should be adhered to, where nothing else is stated.

Headings

The following headings should be used

I am a very large heading (h1 with font-size 28px for desktop); Use me for page headings

I am a large heading (h2); Use me for section headings

I am a medium sized heading (h3); Use me for sub-section headings

I am a relatively small heading (h4); Use me if further sub-division is needed

Text style

Typographical rules for body-text, i.e. the content of an article, ad, news, etc.


Text size and weight

Use me when you need smaller text size: regular weight; or semibold; or bold

Use me when as medium & standard text size: regular weight; or semibold; or bold

Use me when you need larger text size: regular weight; or semibold; or bold 

                    .small-text {
                        font-size: 12px;
                    }
                    .medium-text {
                        font-size: 16px;
                    }
                    .large-text {
                        font-size: 20px;
                    }>
                    .regular-weight {
                        font-weight: 400;
                    }
                    .semibold-weight {
                        font-weight: 600;
                    }
                    .bold-weight {
                        font-weight: 800;
                    }


Text fonts

I am Helvetica; use me as standard, when nothing else is stated

I am google-font "Poppins"; use me for variation, e.g. in an alert


"Fallback" fonts

I am an ordinary sans-serif that almost always work; use me as a fallback font


                    <p class="helvetica-text">I am Helvetica; 
                    use me as standard, when nothing else is stated</p>
                    <p class="poppins-text">I am google-font "Poppins"; 
                    use me for variation, e.g. in an alert</p>
                    <p>I am an ordinary sans-serif that almost always work; 
                    use me as a fallback font</p>
                
                    .helvetica-text {
                        font-family: Helvetica, sans-serif;
                    }
                    .poppins-text {
                        font-family: Poppins, sans-serif;
                    }

Choosing text-size/weight

  • Heading: follow body-text text-size with bold weight
  • Preamble: select larger font-size with semibold weight
  • Body-text: select standard/medium text-size with regular weight

Bold text is to be used for emphasized parts, where italic is not sufficient.


Spacing etc.

Use linebreaks between each sub-section for improved readability.

Body-text color

For ordinary content (ordinary info, ads, this styleguide, etc.), use soft black or yellow

For less happy matters (layoffs, negative economic numbers, etc.), use soft black or red


Header color

Follows the chosen body-text color.


Icon color

Follows the chosen header color.


Background and border colors

  • Body background color, light
  • Body background color, dark
  • Menu, header & footer background color
  • Border color

                    .body-color-light {
                        background-color: rgb(241, 237, 237);
                    }
                    .body-color-dark {
                        background-color: rgb(168, 164, 164);
                    }
                    .menu-color {

                    }

Color combinations

Primarily accessibility reasons requires the following color combinations to be used:

  • Light background - black text
  • Dark background - yellow text
  • Light background - red text

                    #light-black-mix {
                        background-color: rgb(241, 237, 237);
                        color: rgb(50, 50, 50);
                    }
                    #dark-yellow-mix {
                        background-color: rgb(168, 164, 164);
                        color: rgb(251, 250, 157);
                    }
                    #light-red-mix {
                        background-color: rgb(241, 237, 237);
                        color: rgb(141, 25, 25);
                    }
